# Version 2 Roadmap & Notes

## Planned Features
- Support for additional months and years of UK Police data
- Automated data ingestion from multiple folders
- Advanced spatiotemporal clustering (HDBSCAN, DBSCAN with time)
- Predictive modeling by region and crime type
- Animated maps and time series visualizations
- User authentication and role-based access (optional)
- Export results to PDF, Excel, and GeoJSON
- API endpoints for integration with other systems
- Improved error handling and logging
- Performance optimizations for large datasets

## UI/UX Enhancements
- Custom themes and branding
- Enhanced sidebar navigation
- Multi-page dashboard (Streamlit pages)
- User feedback and reporting tools

## Technical Notes
- Refactor data loading for scalability
- Modularize algorithms for easier extension
- Add unit and integration tests
- Document all functions and modules

## Collaboration
- Open for contributions and suggestions
- Please submit issues or pull requests for new features
